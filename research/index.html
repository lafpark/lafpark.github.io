Research
Laurence gets to do interesting things.

Research

Research Interests

  • Information Retrieval - including topics such as Web link analysis, probabilistic latent semantic analysis, latent Dirichlet allocation, evaluation, community tagging, query expansion
  • Data Mining - kernel PCA, non-negative matrix factorisation, spectral graph analysis, co-clustering, VAT, coVAT
  • Machine Learning - support vector machines, conditional random fields, neural networks, self-organising maps
  • Data Compression - managing large sparse or dense data sets for efficient information retrieval
  • Signal Processing - wavelet transforms, Fourier transforms
